Recently Added Sync to Seerr
When Jellyfin reports new library items added, enabling this triggers the "Jellyfin Recently Added Scan" job so that request statuses are immediately updated instead of waiting till next schedule run. Events are debounced, so a bulk import collapses into a single trigger.

Tip: With this enabled, you can set Seerr's recently-added scan Frequency to a much longer interval (60 minutes) to reduce noise.
Wait this many seconds after the LAST item added before firing the scan. Default 60. Range 5 to 3600.
Uses the Seerr URL and API key entered in the Setup section above. If multiple URLs are configured, the first reachable one is used.
